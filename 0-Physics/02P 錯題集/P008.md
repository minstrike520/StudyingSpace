# 解析
個人認為本題敘述不是很清楚，給的內圈焦距不好用。如果題目有說「原本對準內圈的光線在穿越外圈時可視為平行」之類的敘述，就可以算出確切的像距了。
但是，本題**不必算出精確值**。
文中第五行有提到，「有空氣泡的水球，就近似於豎立於O，外圈為匯聚、內圈為發散的薄透鏡」，再加上光線是平行光，物距視為無窮遠，故已可作答(E)。

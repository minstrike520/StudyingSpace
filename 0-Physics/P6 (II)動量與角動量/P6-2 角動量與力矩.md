# 6-2 角動量與力矩
## 角動量與力矩的公式
### 角動量 
$$\vec L = \vec r \times \vec p$$
- 角動量量值：$L = rp\cdot sin\theta$
- 圓周運動的角動量：$L_c = r\cdot mv = mr^2\omega$ 

### 淨力矩 = 角動量時變率 
$$\vec \tau = \frac{d \vec L}{dt}$$

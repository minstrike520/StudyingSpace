# 物質沸熔點的比較
## 常見物質的沸熔點順序
共價網狀物質$\rightarrow$離子化合物$\rightarrow$鹼土金屬$\rightarrow$鹼金屬$\rightarrow$分子化合物
## 沸點比較
氫鍵數/氫鍵有無(正) -> 分子量(正) -> 極性(正) -> 接觸面積(正)
## 熔點比較
氫鍵數/氫鍵有無(正) -> 對稱性(正) -> 分子量(正) -> 極性(正)
## 變因
### 氫鍵與極性
- 氫鍵會阻礙分子的自由移動
### 對稱性
- 對稱性愈大，物質在固態時晶體的堆積就愈緻密，導致在融化時的阻力會更大
### 接觸面積
- 接觸面積愈大，[[C5-8 分子間作用力#分散力|分散力]]愈強，沸點愈高

# 8-5 鹽的分類與命名
- 正鹽：由一種陰離子與一種陽離子形成
- 複鹽：由兩種以上的陰陽離子形成
	- 實例：$\ce{KAl(SO4)2\cdot 12H2O}$(鉀明礬)、$\ce{NaKCO3}$(碳酸鉀納)
- 酸式鹽：多元酸中有部分$\ce{H+}$被取代者，結構像是$\ce{MH_xYO_n}$
	- 實例：$\ce{NaHCO3}$(碳酸氫鈉)、$\ce{NaH2PO4}$(磷酸二氫鈉)、$\ce{Na2HPO4}$(磷酸氫二鈉)
- 鹼式鹽：多元鹼中有部分$\ce{OH-}$被取代者，結構像是$\ce{M(OH)Y}$
	- 實例：$\ce{Mg(OH)Cl}$(氯化氫氧鎂/鹼式氯化鎂)
- 錯鹽：有錯離子的鹽

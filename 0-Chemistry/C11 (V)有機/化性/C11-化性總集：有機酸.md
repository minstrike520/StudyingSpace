#### 2. 脫水
- 需要濃硫酸(脫水劑)作為催化劑 [[C0-總集：催化劑]]
1. 兩個羧酸脫水形成酸酐
2. 分子內脫水形成「五員環」或「六員環」結構，則可得到酸酐 [[柯講-有機化學]]-p87
	- 實例：鄰苯二甲酸 -> 鄰苯二甲酸酐

### 四、氧化
- 普遍而言，有機酸不易被氧化
- [[#甲酸]]、[[#草酸]]具有還原性
	1. 甲酸氧化成二氧化碳
	2. 甲酸與多侖/斐林試劑反應
	3. 甲酸分子內脫水 (製備一氧化碳) [[C10-總表：氣體製備]]
	4. 草酸氧化成二氧化碳
### 五、醯胺生成
[[C11-製備總集：醯胺]]
1. 羧酸 + 胺 -> 羧酸胺鹽
2. 羧酸胺鹽加熱脫水 -> 醯胺

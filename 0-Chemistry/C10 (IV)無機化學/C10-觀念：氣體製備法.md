# 氣體製備
|     | 不需加熱  | 需加熱 |
| --- | ----- | --- |
| 無液體 | 沒這種東西 | A   |
| 有液體 | B     | C   |
## A
固體 + 需加熱
1. 試管要微向管口傾斜，避免冷凝水氣回流污染藥品
2. 關閉時，要**先移除收集器，再移走火源**。否則集氣的水會因空氣冷縮而回流，製造大溫差，致使試管破裂。

## B
有液體 + 不需加熱
1. 薊頭漏斗下口須在水面之下，避免氣體產物從漏斗逸散。

# 電解碘化氫
*可與 [[C9-觀念：電解產物判別法]] 呼應*
陽極：非活性電極 -> 有鹵素陰離子 -> 鹵素陰離子氧化
陰極：陽離子為活性金屬 -> 水還原

陽極反應：
1. $\ce{2I-(aq) -> I_{2}(s) + 2e-}$
2. $\ce{3I-(aq) -> I_{3}-(aq) + 2e-}$

陰極反應：
[[9-1-水相關的氧還]]
- $\ce{2H_{2}O(l) + 2e- -> H_{2}(g) + 2OH-(aq)}$

碘 -> 三碘離子：
*無法觀測到大量碘，因為此反應會發生*
- $\ce{I- + I_{2} -> I_{3}-}$

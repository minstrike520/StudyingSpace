# 9-1 氧化數
## 化合物的氧化數判斷法則
1. $\ce{F} = -1$, $\ce{IA} = +1$, $\ce{IIA} = +2$, $\ce{Al} = +3$
2. $\ce{H} = +1$, $\ce{M_nH_m} \rightarrow \ce{H} = -1$
3. $\ce{O} = -2$, $\ce{O_{過}} = -1$, $\ce{O_{超}} = +2$, $\ce{F_nO_m} \rightarrow O = +2$
4. $\ce{7A} = -1$

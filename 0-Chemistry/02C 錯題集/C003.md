# 訂正
- 列舉所有相關的可能反應：
	1. 烯類加鹵素得二鹵烷
	2. 炔類加鹵素得二鹵烯，再加得四鹵烷
- 原則就是斷一次鍵加兩個鹵素
- 另外，需注意題文有說「**未照光**」，所以烷類鹵化無法進行。

# 化學平衡
- 可逆反應
- 化學平衡 $\rightarrow$ 屬於動態平衡(**有**正逆反應且速率相等)
> [!Note] 等效平衡
> 相異量的反應物+產物組合，在相同條件下可形成相同的平衡狀態（最終反應物、產物量相同），則稱**兩個組合的平衡狀態相同**，此現象稱作等效平衡。
^note1
- 等效平衡題型的解法
	- 將反應物、產物依照反應式全部替換成其中一邊(如四氧化二氮—二氧化氮全部替換為二氧化氮)，則若得其量相等，則即為等效反應。

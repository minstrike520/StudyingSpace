# 實驗：勒沙特列原理
- $\ce{NO2}$的可逆反應 $$\ce{2NO2(g) <=> N2O4(g) + \Delta H}$$
- 關鍵變化：$\ce{NO2}$是紅棕色氣體，而$\ce{N2O4}$是無色氣體，可以透過**觀察系統顏色變化**來得知反應平衡的移動。 [[C-記憶：物質顏色]]
- 測得結果
	- 加熱 $\rightarrow$ 顏色變深；降溫 $\rightarrow$ 顏色變淡
	- 加壓 $\rightarrow$ 顏色變深；減壓 $\rightarrow$ 顏色變淡

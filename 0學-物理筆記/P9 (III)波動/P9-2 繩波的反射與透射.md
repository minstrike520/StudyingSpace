# 9-2 繩波的反射與透射
## 繩波的反射
1. 前後形狀：反射之前跟反射之後，**波形的「前後形狀」固定**。
	- TIP: 有的講義會說成「左右」，但我私心覺得前後比較好理解。
2. 上下形狀：固定端上下倒反，自由端不會顛倒。

## 輕、重繩之間波的反射、透射
- 輕重繩定義：繩子的線密度有差異者。
- 輕重繩的波速差異：$v = \sqrt{\frac{F}{\mu}} \longrightarrow v \propto \sqrt{\mu}^{-1}$，繩子愈重，波速愈慢。

### 輕重繩透射的比較（相對於入射波）
| 變因   | 輕繩->重繩 | 重繩->輕繩 |
| ---- | ------ | ------ |
| 透射波形 | 正小     | 正大     |
| 反射波形 | 反      | 正      |
| 透射波速 | 較慢 | 較快 |
| 透射振幅 | 較小 | 較大 |

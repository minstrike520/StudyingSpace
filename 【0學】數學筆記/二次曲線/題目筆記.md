- 橢圓.標準式.陷阱
	- $$\frac{x^{2}}{3} + \frac{y^{2}}{10} = 1$$
	- 注意標準式中，分母處放的是 $a^{2}$ 跟 $b^{2}$，不要被非完全平方數唬住了！
	- 可能的錯誤：
		- 【錯誤操作】$a = 10$, $b = 3$ (X)
	- 避免方法：
		- 將原本的樣型照抄，不要在第一步就腦補跳步驟。
		- 【作法】$a^2 = 10$, $b^2 = 3$ $\rightarrow$ $a = \sqrt{10}$, $b = \sqrt{3}$ (O)
		- 每一個操作都有其數學意義，不是簡單就一定要跳步驟。(Ext: 同理，不是很多數值計算就要在等式之中展開。)

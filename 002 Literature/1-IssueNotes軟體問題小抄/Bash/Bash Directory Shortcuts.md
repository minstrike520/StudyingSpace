# Bash directory shortcuts

```bash
# Link Shortcut
mkdir <shortcut_link_dir>
ln -s <target_dir> <shortcut_link_dir>

# Add To Enviroment Variables
export CDPATH = <shortcut_link_dir>
cd b   # Switches to <target_dir>
```

# Calc 3.2
Example 1: Differentiate $f(x) = \frac{x}{x-1}$

sol:
$$
\begin{aligned}
 & f'(x) = \lim_{ h \to 0 } \frac{f(x+h) - f(x)}{h} = \lim_{ h \to 0 } \frac{\frac{x+h}{x+h-1} - \frac{x}{x-1}}{h} \\
 & = \lim_{ h \to 0 } \frac{-h}{(x+h-1)(x-1)}\cdot \frac{1}{h} = -\frac{1}{(x-1)^{2}}\ \blacksquare
\end{aligned}
$$

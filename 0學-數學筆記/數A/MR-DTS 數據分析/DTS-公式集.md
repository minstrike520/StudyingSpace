# 數據分析．公式集
## 基礎
- 平均
- 標準差
- 離均差 $= x_k - \mu_X$

## 數據標準化
- 標準化數據 $X' = \frac{X-\mu_X}{\sigma_X}$
- 標準化數據的特性
	1. 平均為零、標準差為一： $\mu_{X'} = 0,\ \sigma_{X'} = 1$
	2. 斜率為相關係數： $m_{X'Y'} = r_{X'Y'} = r_{XY}$
## 最適直線
- 相關係數
	1. $r_{XY} = \frac{S_{XY}}{\sqrt{S_{XX}}\sqrt{S_{XY}}}$
	2. $r_{XY} = \frac{\sum x_k y_k}n$
- 最適直線斜率 $m_{XY} = \frac{S_{XY}}{S_{XX}}$

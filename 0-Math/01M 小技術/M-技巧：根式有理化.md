運用**平方差公式**
$$\begin{aligned}
\sqrt{ a } - \sqrt{ b } = \frac{{a-b}}{\sqrt{ a } + \sqrt{ b }} &  & \sqrt{ a } + \sqrt{ b } = \frac{a-b}{\sqrt{ a }- \sqrt{ b }}
\end{aligned}
$$
分式+根式
$$
\frac{\sqrt{ a }-\sqrt{ b }}{\sqrt{ b }-\sqrt{ c }} = \frac{(a-b)(\sqrt{ b }+\sqrt{ c })}{(b-c)(\sqrt{ a }+\sqrt{ b })}
$$

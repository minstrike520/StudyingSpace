# 題文
方程式$\sin x + log_{\frac 1 9}x^2 = 0$有幾個實根？
# 想法
- 作圖求交點
# 解法
$\sin x = -log_{\frac 1 9} x^2$
	$= -log_9 x^2$
	$= log_3 |x|$  ==注意此步！！在去平方時，要注意平方隱含著abs的「殘留影響」！！== 見[[CRL-KW-NOTE(L)#絕對值的含意/處理方法|圓與直線/絕對值]]
作圖$y = \sin x \land y = log_3 |x|$，有兩交點，得有2相異實根 $\blacksquare$

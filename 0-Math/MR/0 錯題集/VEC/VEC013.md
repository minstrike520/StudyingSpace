# 心得
我認為這一題的關鍵在對題意的清楚程度。
我事後發現，$\vec{OB}$ 的 $B$ 點根本就不會作為點來使用，因此還不如這樣重新表示：
$$
\vec{OC_{n}} = \vec{OA} + n\vec{v}
$$
$A$ 是任意**點**，$\vec{v}$ 是任意**向量**。

## 2.(2)
*KW p7*
### 題文
設數列$<a_n>$滿足遞迴關係式：$\sqrt{a_n} = \frac{\sqrt{a_{n+1}} + \sqrt{a_{n-1}}}2$，其中$n \in \mathbb N \land n\ge 2$，若$a_1 = 24,\ a_2 = 25$，則$a_8$？
### 想法
- 「等差中項」： $a_n = \frac{a_{n+1} + a_{n-1}}2$
	- $a_n - a_{n-1} = a_{n+1} - a_n$
	- $2a_n = a_{n+1} + a_{n-1}$
	- $a_n = \frac{a_{n+1} + a_{n-1}}2$
